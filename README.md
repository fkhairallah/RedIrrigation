# RedIrrigation

[RED]Irrigation is an ESP8266-based IOT that drives RyeManor's Irrigation system

# Version History

## 0.2 Beta
    Migration to Platform.IO

# Hardware Notes

 Zone#, Pin, & Color:
-  ZONE1 5  // RED
-  ZONE2 4  // GREEN
-  ZONE3 14 // BLUE
-  ZONE4 12 // ORANGE
-  ZONE5 13 //YELLOW


// ***BEWARE*** Some ESP8266 pins (15) are set to LOW during boot or flash which activates the relay
